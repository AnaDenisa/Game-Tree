# Game Tree
1. Functionality

This program implements a Game Tree for Tic-Tac-Toc using the Minimax Algorithm.


2. Input format

Input files format:

X	/* whose turn is it */

X - O	  /* the game configuration when the game tree construction begins */

O X -

- - -

3. Run:
./minimax [-c1 |-c2] input_file output_file
